# Clinic Management System

**Clinic Management System** is a desktop application built using **C# Windows Forms** and **MySQL** for managing patient records efficiently in a medical clinic.

---

## Key Features

- **Add New Patient:** Enter patient details including Name, Age, Phone, Address, Notes, and Gender.  
- **Update Patient:** Modify selected patient details such as Name, Age, and Phone.  
- **Delete Patient:** Remove patient records with a confirmation prompt.  
- **Exit Application:** Close the application safely.  
- **Database Backup:** Create a full backup of the database in SQL format with a single click.

---

## User Interface Overview

| Button | Function |
|--------|---------|
| **Save** | Add a new patient record to the database |
| **Update** | Edit the selected patient's details |
| **Delete** | Delete the selected patient record |
| **Exit** | Close the application |
| **Backup** | Generate a SQL backup of the database |

---

## Technology Stack

- **Programming Language:** C#  
- **UI Framework:** Windows Forms  
- **Database:** MySQL  
- **Database Connector:** MySQL Connector/NET  
- **Development Environment:** Visual Studio 2019 or higher  

---

## How to Use

1. Open the project in Visual Studio.  
2. Ensure MySQL Server is running and the `clinic_db` database exists.  
3. Run the application starting from **Form2** (Login form).  
4. Use the buttons to manage patient records as needed.  
5. Create a backup of the database using the **Backup** button whenever necessary.
